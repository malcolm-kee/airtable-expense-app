# airtable-expense-app

Expense Tracking PWA

Features:

1. works in offline.
2. entry will be synced in background if network is offline.

## Make It Your Own!

1. Copy [this Airtable base](https://airtable.com/shr0lUoIRq09R4aIJ). Use the Copy base button at top right.
1. Go to https://airtable.com/api, then choose the base that you just copy. It should include the instruction for you to generate the API Key and the id for that base.
1. Fork this repo and deploy to Netlify.
1. Add the following 2 environment variables in Netlify using the value you get from step 2:

   ```
   AIRTABLE_API_KEY=
   AIRTABLE_BASE=
   ```
